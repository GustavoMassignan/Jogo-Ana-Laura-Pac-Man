# Jogo-Ana-Laura-Pac-Man
Jogo feito/editado por Gustavo Massignan, Laura Guimarães e Isadora Tonolli, onde fizemos uma reeleitura do jogo Pac-Man, o intuito era modificar as habilidades do personagem transformando as frutas em legumes e comidas gordurosas, fazendo assim ele andar mais rapido ou mais devagar, dependendo da comida que ele pegasse.
https://youtube.com/shorts/KilVWOgzAso?si=AdQKMpkHbOo66dE6 --> video do jogo mais explicação da programação
Nome do autor:Anand(Noob_coder) (acho que é esse, pelo menos era este que estava no app do paygame) https://www.pygame.org/project/5666 --> de onde pegamos o jogo
Para jogar o jogo, primeiramente terá de ter o aplicativo Visual Studio Code, em seguida clique em File-Open Folder e selecione a pasta do jogo, que poderá ser baixada apartir deste repositório, em seguida clique em run que jogo sera inicio automaticamente. Também apartir do Visual Studio Code você tem acesso ao código do jogo caso queira fazer alguma modificação.
